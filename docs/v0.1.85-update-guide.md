# Temu 自动化运营工具 v0.1.85 更新说明

发布时间：2026-04-27

## 修复重点

1. 修复批量上品长时间显示“处理中”的问题。
   - 新上品流程现在会把“预热登录态、读取原图、生成白底组合图、上传素材中心、保存草稿”的实时进度同步到任务历史。
   - 上品类长任务主进程超时从短连接改为 12 小时，避免用户机器慢、网络慢时主进程先断开。

2. 修复“草稿保存未完成，请稍后重试”过度泛化的问题。
   - 前端不再把所有草稿失败都压成同一句话。
   - 如果 Temu 已创建草稿但校验失败，会显示真实原因，例如主图不足、父规格未完整保存、SKU 必填项未保存等。

3. 强化高风险商品过滤。
   - 补充水相关、液体、膏体、服装裤子鞋帽、常见 IP/定制图案等关键词。
   - 覆盖花园水管、冰块模具、茶滤/滤茶器、水泵/喷水、服装裤子、卡通/动漫/品牌 logo 等漏网场景。

## 验证

- 已通过 `npm run verify:release`
- 已通过 `npm run build`
- 已通过 `npm run smoke:desktop`
