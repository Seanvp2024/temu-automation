# 同事使用说明

## 1. 安装

1. 双击安装包 `Temu 自动化运营工具-Setup-<version>.exe`
2. 按提示完成安装
3. 确认电脑已安装 Google Chrome 或 Microsoft Edge

## 2. 首次使用

1. 打开 `账号管理`
2. 新增 Temu 账号并完成登录
3. 回到 `数据采集`，执行一次 `一键采集`

建议首次采集完成后再查看：
- `店铺概览`
- `商品管理`
- `商品详情`

## 3. 日常使用

### 数据采集

- 进入 `数据采集`
- 点击 `一键采集全部数据`
- 等待完成后再查看商品、销售、流量等页面

### 商品管理

- 在 `商品管理` 页面筛选在售中、低库存、未加入站点商品
- 点击商品图片可查看大图
- 点击 `查看详情` 查看销售、流量、库存和质量数据

### 批量上品

1. 进入 `上品管理`
2. 上传商品表格
3. 点击 `开始批量上品`
4. 观察页面日志、结果和失败原因

## 4. 更新软件

1. 进入 `设置`
2. 点击 `检查更新`
3. 如果发现新版本，点击 `下载更新`
4. 下载完成后点击 `立即安装更新`

默认会从 GitHub Releases 检查更新，不需要手动填写更新源地址。

如果未配置更新源，也可以直接安装新的安装包覆盖升级。

## 5. 常见问题

### 采集失败

- 先确认账号仍然在线
- 再确认 Temu 页面能正常打开
- 然后到 `日志中心` 看前端报错

### 商品详情没有数据

- 先重新执行一次 `一键采集`
- 如果仍然没有，通常是 Temu 对应数据源暂未返回该商品

### 批量上品失败

- 优先看页面里的 `批量上品日志`
- 常见原因是 AI 生图失败、图片上传失败、类目属性变化或提交核价失败

## 6. 反馈信息时请带上

- 问题页面截图
- 发生时间
- 商品 SKC / SPU / goodsId
- `日志中心` 里的错误信息
